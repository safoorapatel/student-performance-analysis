# Student Performance Analysis 🎓

## Overview
This project explores the factors that influence student exam performance 
using Exploratory Data Analysis (EDA) and Machine Learning.

## Dataset
- 6,606 students
- 20 features including attendance, hours studied, parental involvement, 
sleep hours and more
- Target variable: Exam Score

## Key Findings
- Attendance and hours studied are the strongest predictors of exam performance
- Exam scores follow a normal distribution centered around 67
- The Random Forest model achieved an R² score of 0.69, 
predicting exam scores within 1.12 marks on average

## Tools Used
- Python
- Pandas
- Matplotlib & Seaborn
- Scikit-learn (Random Forest Regressor)

## Project Structure
- `student_performance_analysis.ipynb` - Full analysis notebook
- `StudentPerformanceFactors.csv` - Dataset
